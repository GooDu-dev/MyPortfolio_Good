Websites

เป็นโปรเจคที่ผมได้ทำระหว่างการเข้าค่ายแต่ผมชอบโจทย์ผมจึงอยากมานำเสนอตนับ โดยโจทย์คือ สร้างโปรแกรมคำนวณค่าจอดรถโดยไม่ใช้ if-else

เป็นโปรเจคที่ผมนั่งทำเวลาว่าง ซึ่งผมได้เรียนรู้เกี่ยวกับ RTC Module และ LCD Module ที่นำมาต่อเข้ากับ Arduino และใช้ library เสริมมาสร้างนาฬิกาดิจิตอลครับ

เป็นเกมที่ผมได้เรียนสร้างจาก youtube ตอนที่เกิดวิกฤตการณ์ covid-19 ตอนปี 2019 โดยใช้ Unity3D และสิ่งที่ได้จากโปรเจคนี้คือ การใช้ post processing ในการทำให้ภาพเกมของเราสวยมากขึ้น

เกมนี้เป็นโปรเจคแรกของผมเพราะผมอยากทำเกม 2D โดยผมเลือกใช้ Unity3D ซึ่งทำให้ผมได้เรียนรู้เกี่ยวกับการเคลื่อนที่พื้นฐานและระบบพื้นฐานที่สำคัญของเกมอย่าง ระบบ HP MP ระบบต่อสู้ Animation ฯลฯ

อันนี้เป็นโปรเจคจบค่าย โดยกลุ่มของผมเลือกที่จะสร้างเกมบน Microbit โดยผมเป็นคนคิด Algorithm หลักของเกม และได้ช่วยเหลือกันเองภายในกลุ่มจนผ่านโปรเจคมาได้ครับ

โปรเจคนี้เป็นโปรเจคที่ผมได้ลองทำ เพื่อที่จะได้เรียนรู้วิธีใช้งาน LCD shield กับ Arduino UNO ทำให้ผมได้รู้เกี่ยวกับการนับช่องบน LCD Module เเละการเขียนโปรแกรมควบคุม LCD Module

นี่เป็นรูปที่ผมได้รีบรางวัลชนะเลิศการแข่งเขียนโปรแกรมบังคับหุ่นยนต์รูปแบบทีมในค่าย Golden Robot Arm Camp ซึ่งเรียกได้ว่าเป็นค่ายที่เป็นจุดเริ่มต้นของผมเลย เป็นค่ายที่ผมได้เรียนรู้วิธีการสร้างแขนกล 4DOF ด้วย Servo และ Arduino UNO

รถเก็บขยะอัตโนมัติ โดยส่วนนี้จะเป็นส่วนแขน ที่ไว้คีบถังขยะในโครงงานวิทยาศาสตร์ของผมครับ สำหรับผมเป็นโปรเจคที่ผมสามารถนำไปพัฒนาต่อให้ดียิ่งขึ้นในอนาคตครับ

โปรเจคนี้เป็นอีกโปรเจคที่ผมนั่งทำในเวลาว่างโดย ถังขยะใต้โต๊ะผมที่เหยียบมันพัง ผมเลยต้องการที่จะเปิดถังขยะโดยใช้เเค่ปุ่มกดจะได้ไม่ต้องก้มไปเปิดครับ

รถคันนี้เป็นส่วนฐานของโครงงานวิทยาศาสตร์ของผม โดยผมใช้ Gear Motor ในการเคลื่อนที่ และใช้ Relay ในการบังคับว่าให้เคลื่อนที่ไปยังทิศไหน และติด sensor Ultrasonic ในการ detect วัตถุรอบตัวรถ แล้วเข้า Algorithm เพื่อเช็คว่าเป็นกำแพงหรือขยะไว้ให้แขนเก็บ

เป็นเว็บที่ผมได้ร่วมทำกับเพื่อนๆในค่าย เป็นโปรเจคจบค่ายครับ ซึ่งเป็นเว็บแรกเลยที่ผมได้ทำขึ้นมาด้วย HTML CSS Javascript ครับ

เป็นโปรเจคใหญ่แรกที่ผมหลังจากที่ผมได้ศึกษาการพัฒนาเว็บมา 3 สัปดาห์ ที่ผมได้พัฒนาเว็บไซต์นวัตกรรมเครื่องฉีดวัคซีนอัตโนมัติ โดยผมเป็นคนวางวงจรของเครื่องและ module ที่ใช้ และเป็นคนพัฒนาเว็บนี้ครับ

เป็นโปรเจคที่ผมทำขึ้นมาเล่นๆตอนพักกลางวันครับ โดยผมต้องการที่จะทำเว็บคำนวน BMI ที่สามารถเปิดได้ทั้งในโทรศัพท์และในคอม ทำให้ผมเขียนเว็บแบบ responsive เป็นครับ

เป็นเว็บที่ผมกำลังพัฒนาร้วมกับทีม เพื่อพัฒนาเว็บให้กับเพจ grnSTEM ครับ ซึ่งกาารพัฒนาเว็บให้เพจ genSTEM ทำให้ผมได้ลองใช้ libraly หลายๆตัวเลยครับ เช่น calendar Counter jquery 3.3.1 ฯลฯ ทำให้ผมได้พัฒนาไปอีกขั้นครับ
